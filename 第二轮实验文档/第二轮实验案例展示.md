以眉心伤疤事件为例进行分析：

原文：

```
Jenny(00:40:41): 甩到那个上去那我不知道怎么跑到人家后面去了。跑到人家后面去。
访谈人(00:40:46): 然后就被弄走了？
Jenny(00:40:48): 结果万幸是万幸，他那个铁锹一打打到我的小铲子那个柄子上了，只是我这个疤是我自己那个小铲子。
访谈人(00:40:56): 还好。
Jenny(00:41:02): 崩到我的脸上的，如果真的是铁锹铲到我，你说我还有命吗，就没命了，当时。
访谈人(00:41:11): 非常显。
Jenny(00:41:14): 当时家里人也吓死了，人家挖河工的人也吓死了，你要把人家小孩子给要弄没了，你说还得了吗？我印象特别深，然后我还记得后来是等于是村上面一个叔叔嘛，他就他挖的挖河的那个锹铲到我的铲子的那个叔叔家里面，我还记得送了一篮鸡蛋给我。
Jenny(00:41:43): 就来补，让我来补一补的，知道吗？对人家不害怕嘛，那想个都害怕，我爷爷回家也被我被家里人给说的说你好好把小孩带到那边去干嘛，但我印象很深，就是当时就是他那个铁锹，因为正好甩过来是甩到我的铲小铲子上面的那个响声很大。
访谈人(00:41:45): 也挺好的。就还安抚你一下。
Jenny(00:42:11): 把整个工地人都听见了，都吓死了，大家都呆住都呆掉了，都赶紧蹦过来，自己现在我没有意识的，我当时可能昏过去了，也不知道是没有意识了，后来才知道我妈妈后来给我赶紧找那个自行车把我背的。

Jenny(00:42:56): 医那个公社医院出来的时候妈妈给我买了那时候叫人家乡下卖的，现在你们肯定没见过的那个叫赖葡萄。长的那个金黄色的又发一点红吧，红金黄色的那种赖葡萄，我记得当时不知道买了两三个，就我妈妈相当于买了一个哄我的嘛，别的不记得就记得赖葡萄了。
Jenny(00:43:27): 还有几对，然后就是还有记得就是那个叔叔他家里邻居相当于村上面的送了一篮鸡蛋给我补补。
访谈人(00:43:28): 也是当时比较想吃。
```

Base方法：
```
我眉心的伤疤也有段故事。那时冬天农民都要出河工修水利，爷爷带我去工地玩，给了我个小铲子。没想到挖河工人的铁锹甩过来打到我的小铲子上，铲柄崩到脸上留下了疤。幸好没直接打到，不然命都没了。后来那位叔叔家还送了一篮鸡蛋给我补身体，从公社医院出来时，妈妈还给我买了赖葡萄哄我。
```

Few-Shot方法：
```
我眉心有道疤，是小时候爷爷带我去挖河工地玩时留下的。大人们用铁锹挖土修水利，我拿着小铲子跟在后面玩。有个人转身甩土时，铁锹打在我的小铲柄上，崩起的木屑划伤了额头。万幸的是没被铁锹直接打到，否则后果不堪设想。后来那家人送来一篮鸡蛋给我补身体，从公社医院缝针回来，妈妈还给我买了赖葡萄哄我。
```

CoT方法：
```
我眉心有道疤，是小时候留下的。那时冬天要出河工修水利，爷爷带我去工地玩，给了我个小铲子。我在一个挖河工的人身后玩，他转身甩土时，铁锹打到我的小铲子上，崩起来划伤了额头。万幸的是没直接打到身上，否则命都没了。后来那个叔叔家还送了一篮鸡蛋给我补身体。妈妈带我去公社医院缝针，出来时给我买了赖葡萄，那金黄色的果子到现在都记得。
```

Combined方法：
```
我眉心有道很长的疤，是小时候留下的。那时冬天农闲，每家都要出劳力去挖河，叫"出河工"。爷爷身体不好不能去，就在家带我。有一次挖河工挖到我家后面，爷爷带我去玩，给我个小铲子。我在工地乱跑，一个叔叔转身甩铁锹时，铁锹打在我的小铲子上，崩起来的铲子划伤了我的眉心。万幸的是没直接打到身上，否则命都没了。后来那叔叔家还送了一篮鸡蛋给我补身体。从公社医院缝针出来，妈妈给我买了赖葡萄哄我。
```


原始文本口语化、详细，事件逻辑自然：时间为冬天农闲（非农忙，出河工修水利）；人物包括爷爷（身体不好，在家带孩子，不能出工）、奶奶（曾出工做饭，隐含家庭背景）、挖河工人（叔叔，甩铁锹导致事故）、妈妈（带去医院缝针，买赖葡萄哄）；逻辑为因果链（爷爷带去玩→给小铲子→小孩乱跑到后面→铁锹甩到小铲子柄上崩伤→万幸非直接打到→大家吓坏→叔叔送鸡蛋补、妈妈哄）。

Base方法：逻辑不自然，“那时冬天农民都要出河工修水利，爷爷带我去工地玩”，直接跳跃，丢失爷爷“身体不好就在家带我”的因果（为什么爷爷带孩子），以及行为的逻辑（爷爷从带孩子转为直接“带去工地玩”，非原文“河工就挖到我们家老家后面...爷爷就带我过去玩”）。

Few-Shot方法中问题还比较多：原始中人物互动“爷爷就带我过去玩...给了我一个小铲子...我等于说我也不懂，我小孩子哪懂就跟着去玩，就看着人多的地方就喜欢人多的地方嘛，结果人家在那挖...我不知道怎么跑到人家后面去了”，改写为“大人们用铁锹挖土修水利，我拿着小铲子跟在后面玩。有个人转身甩土时，铁锹打在我的小铲柄上，崩起的木屑划伤了额头。”丢失爷爷的核心人物作用（爷爷带去、给铲子），扭曲为“我拿着小铲子跟在后面玩”（强加“跟在后面”的位置逻辑，原始是“跑到人家后面去”，非固定跟在）。另外“崩起的木屑”（原始是“铲柄崩到脸上”，非木屑）扭曲事件细节。

相比之下Few-Shot、CoT方法都或多或少有些问题，丢失较多信息（农闲、爷爷身体不好带我、挖到家后面），而Combined方法表现最好。

Combined方法：原始中背景“那时冬天农闲，每家都要出劳力去挖河，叫'出河工'。爷爷身体不好不能去，就在家带我。有一次挖河工挖到我家后面”，改写相同，完整保留“冬天农闲”“爷爷身体不好不能去，就在家带我”“挖到我家后面”。人物互动“我在工地乱跑，一个叔叔转身甩铁锹时，铁锹打在我的小铲子上，崩起来的铲子划伤了我的眉心”，保留“爷爷带我去玩，给我个小铲子”“我不知道怎么跑到人家后面去了”。“万幸的是没直接打到身上，否则命都没了。后来那叔叔家还送了一篮鸡蛋给我补身体。从公社医院缝针出来，妈妈给我买了赖葡萄哄我”，保留因果链（崩伤→万幸→送鸡蛋→缝针→哄），描述准确。


### 2. 对情感的理解问题（对情感的解读比较生硬，美化负面情感等）

上一轮实验中发现大语言模型有美化负面情感的倾向是因为对于“爸爸妈妈是因为奶奶出河工认识的”这个地方，省略了一些负向内容。

上一轮报告
```
原文：水利的这种沟渠挖出来都是人工挖出来的，都是就是农民，就是每次到了不农闲的时候，每家都出劳工，你家没有劳工，比如说我印象当中我奶奶还去她我们家，我爷爷身体不好，没有人出河工怎么办？就我奶奶去。工地去干什么，去帮人家做饭。因为挖河工的人要吃饭嘛，他就等于说你去做饭也算你出工嘛？
…
记得这个我就讲什么，就是那个时候，我觉得就是我们国家很长时间，就是在解放后吧，那种挖河工。曾经是每年就是占农民的，一个是相当于就相当于出劳力的一个很大的一个部分，因为基本上每年冬天，大家都会就是去挖河工，当然了，现在想也不全是不好的，其实有很多的。

DS：每年冬天，农民要去挖河，叫“出河工”。那是国家水利建设的重要部分，全靠人力。我爷爷奶奶都去过，奶奶是去帮厨。我爸妈也是在挖河时认识的。
```

但在本次实验中，发现四个prompt生成的文本中均省略了这一部分，

Base:
```
无
```

Few-Shot：
```
出河工是那个年代农民的重要劳动，我爸妈就是在奶奶出河工时相识的。
```

CoT:
```
我爸妈能认识，也是因为奶奶去出河工时到了外婆家那边。那时候出河工也是人们交流的重要方式。
```

Combined:
```
无
```

主要是这一点也确实不是很重点的地方，在改写时将这一句省略了，于是对于这一点“对于情感的理解问题”不再在此处进行分析。
